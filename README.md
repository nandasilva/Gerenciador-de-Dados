Gerenciador de Dados
====================

Sistema para gerenciar dados para serem compartilhados com o setor de T.I das empresas, dados como FTP, senhas de painéis, emails, etc.

### Front-end

* HTML5
* CSS3
* Javascript

### Back-end

* PHP 5.3 +
* MySQL 5.5 +

### Tecnologias

* Twitter Bootstrap
* Mustache
* LESS
* ZeptoJS
* Slim Framework
* IDIORM
* Composer

### Arquiteturas

* REST + MVC
